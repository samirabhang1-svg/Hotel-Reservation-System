 🏦 Bank Management System

A console-based **Bank Management System** developed using **Core Java, JDBC, MySQL, and Object-Oriented Programming (OOPs)**.
This project simulates real-world banking operations and demonstrates the practical implementation of database connectivity,
transaction management, and user authentication.

🚀 Features

* User Registration
* User Login Authentication
* Bank Account Creation
* Balance Inquiry
* Credit Money
* Debit Money
* Fund Transfer Between Accounts
* Secure PIN Verification
* Logout Functionality

🛠️ Technologies Used

* Core Java
* JDBC
* MySQL
* OOPs
* Maven
* Git & GitHub

📚 Concepts Covered

* JDBC Connectivity
* CRUD Operations
* PreparedStatement
* ResultSet Handling
* Exception Handling
* Transaction Management
* Commit & Rollback
* Database Design
* Object-Oriented Programming
 🔄 Transaction Handling

A key feature of this project is the implementation of transaction management during fund transfers. Multiple database
operations are executed within a single transaction to ensure data consistency. If all operations succeed, the transaction
is committed; otherwise, rollback is performed to prevent partial updates and maintain database integrity.
